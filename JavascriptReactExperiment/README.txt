1. Install node js which also installs npm pakage manager

2. Create the react nextjs app:

https://nextjs.org/learn/basics/getting-started/setup

3. To run 

npm run dev

and browse to: http://localhost:3000 

4. For tests see https://devhints.io/jest

npm install --save-dev jest babel-jest
 + other packages. See Package file.

Then run

npm test
